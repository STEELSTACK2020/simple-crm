One-Time Setup (~10 minutes)

  1. Go to Google Cloud Console

    console.cloud.google.com

  2. Create a project (top dropdown > New Project) or select existing
  3. Enable the Google Analytics Data API

    Go to this link and click Enable

  4. Set up OAuth Consent Screen

    Go to APIs & Services > OAuth consent screen (or "Google Auth Platform" > "Branding")

    Choose "External", fill in App name & your email, click through to save

  5. Add yourself as a Test User

    Go to "Google Auth Platform" > "Audience" (or OAuth consent screen > Test users)

    Click "+ Add Users" and enter your Gmail address

  6. Create OAuth Credentials

    Go to APIs & Services > Credentials > Create Credentials > OAuth client ID

    Application type: Web application

    Under "Authorized redirect URIs" click + ADD URI and enter:

    http://localhost:5000/traffic/oauth/callback

    Click Create

  7. Copy Client ID and Client Secret below

    Client Secret usually starts with GOCSPX-
